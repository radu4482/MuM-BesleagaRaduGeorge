Scop

Scopul aplicatiei Mum este de a evalua productiile muzicale , respectivii artisti, fiind organizati pe diverse categorii (artist , gen muzical).

Tehnologiile folosite

HTML si CSS

HTML

Am folosit tehnologia HTML pentru a crea interfata grafica si pentru a putea crea o anumita legatura cu clientul(utilizatorul).

HTML se poate genera direct utilizând tehnologii de codare din partea serverului cum ar fi PHP, JSP sau ASP. Multe aplicații ca sistemele de gestionare a conținutului, wiki-uri și forumuri web generează pagini HTML.

CSS

Am folosit limbajul CSS pentru a putea a imbunatatii aspectul vizual al proiectului , prin colorarea , modificarea formelor si alte tehnici digitale de manipulare a obiectelor din cadrul codului HTMl.

Avand in vedere faptul ca codul HTML nu dispune de o paleta mare de optiuni grafice, am considerat ca limbajul CSS imi va fi de folos.

JavaScript

Este esentiala pentru crearea paginilor web dinamice , incat am folosit aceasta tehnologie in locul clasicului REFRESH.

Spre exemplu am folosit in cadrul butonului de comentarii si in cadrul butonului de like tehnologia suitei AJAX. Incat am putut sa returnez pe pagina principala requesturi catre server ce au fost posibile cu ajutor unor scripturi PHP care vor afisa raspunsurile pe partea de front-end.

PHP

Am putut face posibil mare parte din proiect cu ajutorul fisierelor si tehnologiei PHP, prin care am putut face requesturi catre baza de date si gestiona raspunsurile, prelucra datele si afisarea informatiei in anumite componente din cadrul codului.

Arhitectura aplicatiei - back-end

Pe partea de baza de date , am folosit phpmyAdmin , crand o multitudine de tabele (artist, comment, like, song , user)

User: contine id, admin (0 in cazul in care este utilizator obisnuit, 1 in cazul in care este admin) , username,password si email.

Song: contine id, artist_id , name, path si gen( tipul muzical al melodiei echivalenta cu cea a artistului)

Like: contine id, song_id, user_id

Comment: contine id, song_id, user_id, comm(mesajul utilizatorului), time_upload(momentul in care a fost incarcat mesajul)

Artist: contine id, name si gender(tipul muzical predominant al artistului)

Am folosit o multitudine de fisiere de tipul php pentru a putea crea functiile dorite si pentru a putea afisa valorile corecte luate si procesate din baza de date.
